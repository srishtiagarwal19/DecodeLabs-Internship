# Rule-Based AI Chatbot

## Objective
Create a simple rule-based chatbot that responds to predefined user inputs.

## Features
- Greetings handling
- Exit command handling
- Time display
- Continuous loop
- Input sanitization
- Fallback response for unknown inputs

## Technologies Used
- Python

## How to Run

```bash
python chatbot.py
```

## Sample Commands
- hello
- hi
- how are you
- what is your name
- what's your name
- help
- thanks
- time
- bye
